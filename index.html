Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

There are a couple of ways to do this. I would start to explain the most common security treats that is out there in short sentences:

  • Cross-site scripting(XSS), a malicious script that has the potencial to steal the user information from their site.
  • SQL Injection, malicious data sent from the user side that can corrupt tha database.
  • Cross-site request forgery(XSRF), tricking users to click a link for profit gain.
